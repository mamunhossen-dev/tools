# Mamun's Tool Hub

A comprehensive workspace of high-utility web tools designed for professionals. This repository serves as a central dashboard for various calculators and estimators I have developed.

[Live Demo](https://mamunhossen-dev.github.io/tools)

## 🛠️ Included Tools

### 🏗️ Civil & Construction
* **Brick & Mortar Calculator:** Estimate bricks, cement, and sand for walls including plastering deductions.
* **Circular Tank Estimator:** Specialized volume calculations for masonry in circular structures.

### 💰 Finance & HR
* **Salary Breakdown:** Calculate gross, net, PF, and bonuses.
* **Increment Tracker:** Project career growth and salary hikes based on percentages.

## 🎨 Design Features
* **Modern Dashboard:** A high-contrast Dark/Light UI inspired by professional SaaS platforms.
* **Fully Responsive:** Optimized for field use on mobile devices and office use on desktops.
* **Category Filtering:** Easily toggle between Engineering and Finance modules.

## 🚀 Deployment
This hub is hosted via **GitHub Pages**. To add a new tool:
1. Create the tool in its own repository or sub-folder.
2. Add a new `tool-card` div to the `index.html` of this Hub.
3. Update the `data-category` for filtering logic.
